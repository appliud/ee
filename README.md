## 实验项目：my_learning（第一次实验）

### 目录结构
- **java/**  
  源代码目录，包含所有Java源代码文件。

- **java/resources/**  
  资源目录，存放配置文件、静态资源等。

- **test/**  
  测试目录，包含单元测试和其他测试脚本。

### 使用依赖
- **Lombok**  
  减少样板代码，提升开发效率。

- **MySQL**  
  数据库管理系统，用于数据存储和管理。

- **JUnit**  
  单元测试框架，帮助验证代码的正确性。

- **MyBatis**  
  持久层框架，简化数据库操作。
